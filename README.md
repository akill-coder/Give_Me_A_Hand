# GiveMeAHand
Hand as a pointing device

Install Python.

Modules used:
   1. cv2
   2. wx
   3. pynput
   4. numpy
         Download them if u dont have them already.
         
         
 Open any python interpretor, copy paste the main.py file and run the python project
 and you are done.
